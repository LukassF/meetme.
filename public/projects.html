InForClimate

  • React JS

InForClimate is a weather checkout app for either a phone or a desktop. Its features range from providing information concerning the current weather conditions in a location chosen by the user, to acquiring forecast data up to five days ahead. Taskify

  • React JS
  • Redux
  • Bootstrap

Taskify is an application designed mainly to be used as a custom ToDo list. It has all of the features typical for these kind of projects, such as an option to list current tasks, as well as the already completed ones. Moreover it introduces a deadline system and a stylish drag and drop list for completed and failed tasks, with an option to change their status.

Peace.io

  • Javascript
  • THREE JS
  • HTML

Peace.io serves as a 'break away from the real world problems' type of game. It doesn't have any particular goal.. besides dropping boulders into the void with the tumbleweed you are in control of. The game also offers the time of day feature where you can experience a different looking environment during noon, sunset or in the middle of the night.
